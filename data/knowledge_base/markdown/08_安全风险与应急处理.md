# 08_安全风险与应急处理

## 1. 适用范围

本文整理火灾、直流拉弧、设备过热、外部环境火源、触电风险、组件爆裂和极端天气等安全风险。内容来自当前资料中的安全事故和应急处理相关材料。

## 2. 火灾风险

资料中指出，光伏电站火灾隐患复杂，着火源包括直流拉弧、设备过热和外部环境因素。

### 2.1 直流拉弧

资料中说明，光伏系统直流侧运行电压通常在 600V-1500V 之间。当线路连接端子松动、接触电阻增大，或绝缘层老化破损时，就会产生直流电弧。直流电弧温度可达 3000℃ 以上，资料中另有表述为 3000 到 7000℃，可引燃线缆绝缘层和周围可燃物。

常见因素包括：

- MC4 连接器压接不牢。
- 线缆绝缘层物理损伤。
- 啮齿动物咬断线缆绝缘皮。
- 端子松动、接触不良。

### 2.2 设备过热

资料中提到，逆变器、汇流箱等功率转换设备运行时会产生热量。如果散热风扇故障、通风通道堵塞或环境温度过高，会造成内部热量积聚。温度持续上升会使电容、电感、功率器件等元器件绝缘性能下降，严重时金属部件熔化并引发火灾。

夏季高温巡检时，如果发现汇流箱外壳温度异常，需要立即排查隐患。

### 2.3 外部环境因素

外部环境风险包括：

- 组件下方和周边杂草干枯，形成易燃物。
- 组件表面鸟粪、灰尘等污染物导致局部热斑。
- 鸟类、啮齿动物在设备箱内筑巢，干草遇到电气火花容易被点燃。

## 3. 防火措施

资料中列出的防火措施包括：

- 规范管理消防器材。
- 做好消防器材定期检查。
- 提升现场人员防火和救援能力。
- 做好电缆及电缆沟防火封堵。
- 排查和整治现场防火隐患。
- 电缆沟道隔开。
- 使用阻燃电缆降低火灾风险。

## 4. 组件破裂与安全

组件破裂可能影响整个阵列发电效率，也可能带来高空坠落和人员安全风险。资料中列出的处理方式包括：

- 断电操作。
- 记录并报告故障。
- 安全清理。
- 更换损坏组件。
- 定期检查与维护。

组件破裂原因包括石子、树枝、冰雹、运输过程碰撞、安装过程操作不当、运维不规范和组件自身缺陷。

## 5. 触电和极端天气风险

资料中的运维风险还包括触电风险、极端天气作业、设备故障停机和电网限电等。雷雨、高温、大风、冰雹、暴雪等极端天气后，应重点检查组件、支架、线缆、防雷和设备外观。

## 6. 安装与维护安全禁忌

组件安装安全要求包括：

- 施工过程严禁踩踏电池板。
- 严禁挤压或用尖锐物敲打、碰撞、刮划组件。
- 严禁将安装工具随意摆放在组件上。
- 严禁触摸组件串金属带电部位。
- 严禁同一片组件连接线正负极插头对接。
- 严禁采用接线盒或连接件方式搬抬组件。
- 严禁在下雨、下雪、大风天气安装光伏组件。
- 严禁单人搬运组件，应两人抬运并轻拿轻放。
- 现场损伤组件禁止使用。
- 施工现场已开箱组件应正面朝上平放，底部垫木质托盘和电池板包装物，严禁立放、斜放或悬空，严禁背面直接暴露在阳光下。

逆变器安装和维护安全要求包括：

- 实施配线及维修时，应切断机柜内所有开关。
- 机柜通风应保持良好，进出口通畅。
- 柜体接地端子必须良好接地。
- 严禁正负极性接反。
- 为防止触电危险，严禁非专业人员私自打开设备机柜门板。
- 设备应远离火源，不能安装在易燃易爆环境中，也不能安装在没有防火保护的设备旁边。
- 接线时应保证所有接线柱和螺栓紧固，接触良好。
- 系统带电时不要连接或断开导线和接线端子。
- 未接入外部电源时，设备内部仍可能存在高电压，严禁触摸。
- 维修维护逆变器必须由专业技术人员操作。
- 维修维护人员需穿绝缘鞋、戴绝缘手套等安全防护设备。
- 维修逆变器时，必须在电源断开 25 分钟之后，并使用测试仪器确认逆变器放电完成、低于 30V 后再操作。
- 维修维护变流器时，必须断开所有开关，确保整个逆变器不带电。

# 04_SVG与无功设备故障

## 1. 适用范围

本文整理 SVG 运行环境、停止工作、功率单元故障、电网电压异常保护、通信故障、运行中停机等内容。内容来自当前资料中两篇 SVG 相关材料。

## 2. SVG 运行环境

资料中指出，SVG 是光伏电站无功补偿的关键设备，关系到电网供电质量和电站发电效率。SVG 工作环境复杂，设备精密，容易受多种因素影响。

运行环境风险包括：

- 电网电压波动：SVG 需要稳定电网电压，电压超范围或频率异常可能使控制系统出问题，严重时触发保护或损坏设备。
- 温度与湿度变化：IGBT 模块、电容、散热系统等对温湿度敏感，高温和潮湿会加速老化。
- 灰尘与污垢：户外或半封闭环境中长期积灰，会影响散热，严重时触发过热保护。
- 设备老化与维护不到位：IGBT、电容、驱动板等元件长期使用会老化或接触不良。

## 3. SVG 装置停止工作

资料中列出的可能原因：

- 控制电源故障，例如电源线断开、电源模块损坏。
- 冷却系统故障，例如风扇电机不转、散热片积灰。
- 系统电压不稳定或完全断电。
- 控制柜电路板故障，输出控制信号异常。
- 连接变压器故障，例如绕组短路、绝缘损坏。

处理方法：

- 检查控制电源、电源开关、电源线和电源模块指示灯。
- 检查冷却系统风扇是否正常转动，有无异常噪音。
- 检查散热片是否积灰，必要时清理滤网。
- 用电压测量设备检测系统电压是否在允许范围内。
- 检测控制柜电路板输出信号。
- 检查连接变压器绕组电阻和绝缘性能。

## 4. 功率单元故障

故障表现：

- 功率输出不稳定。
- 运行温度快速上升。
- 保护功能频繁启动。

可能原因：

- IGBT 模块、电容、触发板等内部元件老化或损坏。
- 散热器积灰堵塞。
- 散热风扇不转。
- 配套空调制冷失效。
- 控制柜驱动信号错误或丢失。
- 光纤连接松动或损坏。

处理方法：

- 检测功率单元内部元件，损坏时更换。
- 清理散热器、风扇等散热部件。
- 修复或更换散热系统故障设备。
- 检查控制信号和光纤连接线路，保证信号稳定。

## 5. 电网电压异常保护

故障现象：

- SVG 无法正常调节无功功率。
- 系统界面显示“电压异常”。
- 系统界面显示“功率因数不达标”。

可能原因：

- 电网电压大幅波动，超过 SVG 调节范围。
- SVG 控制系统与电网之间通信异常，无法接收实时电压数据。
- 逆变器输出电压不稳定，导致 SVG 无法按电网需求响应。

处理方法：

- 用专业设备测量电网电压是否在 SVG 正常调节范围内。
- 必要时调整 SVG 相关参数。
- 检查控制柜和后台之间通信线路是否松动或损坏。
- 对逆变器输出电压进行调试，使其稳定在规定范围内。

## 6. 通信故障

故障表现：

- SVG 与后台监控系统通信中断。
- 设备状态无法上传。
- 控制指令无法下达。

可能原因：

- 通信电缆未插紧、外皮破损或接口接触不好。
- 通信协议设置错误，设备和监控系统规则不一致。
- 后台监控系统故障，例如软件崩溃或服务器死机。

处理方法：

- 检查通信电缆两端接口，确保连接牢固。
- 电缆破损时更换或重新接好松动线头。
- 对照设备说明书和监控系统配置文档，确认通信协议参数正确。
- 查看后台监控系统运行日志和错误提示。

## 7. 运行中停机

另一份 SVG 材料中提到，SVG 运行中突然停机，一般表现为控制柜“就绪”指示灯熄灭，无法继续运行。

可能原因：

- 电网停电或电压崩溃，SVG 启动自我保护。
- 控制柜电源电压波动或模块故障。
- 电路板信号输出异常，导致 SVG 误判工况。

处理顺序：

- 先检查网侧是否停电，排除外部干扰。
- 网侧正常时，再检查控制柜、电源和电路板信号。

## 8. 无功补偿装置安装检查

无功补偿装置用于提高用电自然功率因数，并根据并网侧电压变化投入或切除，防止无功电力倒送。

无功补偿装置安装流程包括：

- 施工准备。
- 基础复测。
- 电抗器安装。
- 避雷器安装。
- 隔离开关安装。
- 真空断路器安装。
- 电容器系统安装。
- SVG 功率单元安装。
- 设备之间电气连接。
- 质量验收。

电抗器基础和支架安装检查要点包括：

- 基础轴线偏移量和基础杯底标高偏差应在规范允许范围内。
- 相间中心距离误差不大于 10mm，预埋件中心线误差不大于 5mm。
- 支柱绝缘子标高误差控制在 5mm 内。
- 设备支架安装后的标高、垂直度、轴线偏差、顶面水平度和间距偏差应满足规范。

电抗器安装要点包括：

- 电抗器垂直安装时，各相中心线应一致。
- 吊装时应使用专用吊具，穿过吊码起吊。
- C 相安装后再安装 B 相和 A 相。
- 电抗器垂直度应均匀分配于所有支柱绝缘子上。
- 围栏或围网应一处开口，并用非金属材料联接断口，避免形成闭合环流。
- 围栏到线圈中心距离不小于 1.1 倍线圈最大外径。

避雷器安装要点包括：

- 基础中心距离误差、高度误差、预埋件中心距离误差均应不大于 10mm。
- 安装过程中不得翻转、碰撞和强烈振动。
- 安装均压环时不能碰损，不能倒装。
- 使用清洁剂、润滑剂、密封脂和擦拭材料应符合产品技术规定。
- 绝缘子应清洁完整。
- 设备铭牌、安装编号和相色标识应齐全。

电容器系统安装要点包括：

- 电容器安装前外观应无破损、锈蚀和变形，金属构件无明显变形、锈蚀。
- 每层电容器间距不小于 100mm，排距不小于 200mm。
- 电容器底部距地面不小于 300mm。
- 不得安装妨碍空气流通的水平层间隔板。
- 电容器连线应采用软导线。
- 电容器外壳与电位固定点应良好相接，布置应使铭牌向外，便于检查。
- 电容器组接通前应与放电线圈可靠连接。

SVG 功率单元安装和存放要求包括：

- 高压静止无功发生器应存放在空气流通处。
- 存放温度示例为 -40℃ 到 +70℃，空气最大相对湿度不超过 90%。
- 存放期间应避免阳光直射，防止水浸、雨淋和腐蚀性气体侵蚀。
- 安装时应按现场安装图纸进行。
- 柜体应固定牢固并可靠接地。
- 设备内接地端子应连接至室内大地。
- 柜体间应相互连接形成整体。
